태그 
            
    
                
  1. p : 문단
    2. 

                
  2. br : 줄 띄우기
    3. 

                
  3. strong : 진하게
    4. 

                
  4. h1~h6 : 제목
    5. 

                
  5. img : 이미지를 삽입
    6. 
                    - src="~": 위치 (파일명이 ~이다)

                
  6. li : 리스트
    7. 

                
  7. ul : 리스트들을 그룹화함 (li의 부모 태그)
    8. 

                
  8. ol : 번호가 붙는 ul (ordered list)
    9. 

                
  9. center : 가운데 정렬
    10. 

                
  10. pre : 텍스트 그대로 나타내기 
    11. 
                    - 띄어쓰기, 줄띄우기 등 표현에 편리
                    - 특수기호는 표현 안 됨
                    - 글꼴이 설정되어 있어 글자 크기가 바뀔 수 있음

                
  11. span : 어떠한 의미도 없는 (디자인 용으로만 쓰는) 태그 
    12. 

                
  12. div : span 태그의 display:block 버전 
    13. 

                
  13. font-family : 글꼴 
    14. 

                
  14. a : 링크 걸기
    15. 
                    - href="~" : 위치
                    - 속성 target
                    - target="_blank" : 새 탭에서 열기

                
  15. <!-- 어쩌구 --> : 주석 처리
    16. 

                
  16. input : 뭔가를 넣는 태그 (클라이언트가 값을 입력할 수 있음)
    17. 
                    - type 속성 : button, text 등 여러 유용한 타입 있음
                    - value 속성 : 초기값
                    - onclick 속성 : 클릭했을 때 일어나는 일 (값으로 JS가 들어옴)
                    - onchange 속성 : 내용이 변했을 때 일어나는 일
                    - onkeydown 속성 : 키보드를 눌렀을 때 일어나는 일

                
  17. script : 'JS가 들어갈 것이다'라는 뜻의 태그
    18. 
                (CSS의 'style'같은 태그)
